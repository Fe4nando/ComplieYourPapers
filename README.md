# gmakpaperport 
Introduction
This application is a Streamlit-based web app that allows users to download and merge CAIE IGCSE and A-Level past papers efficiently.
Users can select subjects, paper variants, start year, and number of past years. The system automatically downloads the selected papers, organizes them, and optionally merges them into clean, structured PDFs.
The app uses multithreaded requests for faster downloading and provides an interactive web interface powered by Streamlit.

Operating Systems it works on:
Windows
MacOS
Linux
(Any OS that supports Python and Streamlit)

How to Run
Open terminal / command prompt
Navigate to the project folder
Run:
streamlit run app.py
Open the local browser link provided (usually http://localhost:8501)

Features
Download IGCSE and A-Level past papers
Select Question Paper or Marking Scheme
Choose start year and number of past years
Merge papers by paper code (11, 12, 21, etc.)
Optional custom cover page generation
Multithreaded downloading for improved speed
Interactive web interface
